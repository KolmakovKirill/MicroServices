<h1>NotificationService</h1>
Проект сделани в рамках студенческого обучения. 
<p>Микросервисный сервис уведомлений приниает запросы по отправке сообщений и отправляет сами сообщения по различным каналам (Email, SMS, Push, Messenger). 

<h1>Репозиторий</h1>
<p>CommonShared - 
<p>Gateway - 
<p>EmailSender, PushSender, SmsSender, MessengerSender - реализуют отправку сообщений через соответствующие каналы
